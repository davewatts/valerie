    • valerie

    A validation library for Knockout.

    Project maintained by egrove Hosted on GitHub Pages — Theme by mattgraham

    Features include:

    • value parsing and formatting
    • property validation
    • model and sub-model validation
    • rule chaining
    • value bindings
    • CSS bindings
    • static and dynamic validation summaries
    • easy to extend
    • easy to localise
    • designed to work with older browsers (just like Knockout itself)
